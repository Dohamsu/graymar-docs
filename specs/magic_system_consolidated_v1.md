## 1. 마법 세계관 기본 원칙 (Magic Worldbuilding)

마법은 존재하지만 극히 제한적이며, 정치 권력과 직접 연결된 자원이다.

- 마법은 일상적이지 않다.
- 일반 민중은 마법을 거의 보지 못한다.
- 마법은 왕실/기사단/특정 귀족 가문만 사용 가능.
- 마법은 정치적 자산이다.

### 1.1 마법의 희소성 (Rarity)

- 마법은 소수의 권력자만 접근 가능한 자원
- 일반인에게 마법은 소문 수준이며, 미신과 구분이 어려움
- 마법 사용자는 경외와 두려움의 대상

### 1.2 마법의 정치적 성격 (Political Nature)

마법은 단순한 능력이 아니라 정치적 도구이자 상징이다.

- 균형을 깨는 변수
- 쿠데타 도구
- 정통성 상징
- 암살 수단

## 2. 마법 보유 세력 (Magic Holders: Royalty / Knights / Nobles)

### 2.1 왕실

- 의식 마법
- 혈통 기반 마법
- 봉인/계약 관련 능력

> 정통성 유지 수단으로 기능

### 2.2 기사단

- 전투 강화 마법
- 축복/방어 의식
- 성물 기반 능력

> 군사적 우위 요소로 기능

### 2.3 귀족 일부

- 고대 계약
- 유물 기반 능력

> 그러나 불법적이며 비밀 유지 필요

---

## 3. 플레이어 마법 해금 경로 (Player Magic Unlock Paths)

### 3.1 기본 철학

- 플레이어는 초기 비마법 인물
- 플레이어는 혈통 각성형이 아님
- 마법은 "능력"이 아니라 "권한"에 가깝다
- 마법은 정치적 연결의 결과
- 특정 스토리 조건 충족 시 제한적 마법 사용 가능

### 3.2 해금 조건 (Unlock Conditions)

마법 사용은 다음 중 하나 이상 충족 시 가능:

- 왕실과의 계약
- 기사단 성물 접근 권한 획득
- 특정 귀족 가문과 비밀 동맹
- 금지된 고대 유물 확보

## 4. MAGIC_ACTION 카테고리

마법은 일반 스킬이 아니며, 별도 카테고리 `MAGIC_ACTION`으로 정의된다.

| Action | 설명 |
|--------|------|
| `ARCANE_STRIKE` | 마법 기반 공격 |
| `WARD` | 마법 방어/보호 |
| `SEAL` | 봉인 |
| `BIND` | 구속/속박 |
| `REVEAL_TRUTH` | 진실 간파 |
| `ROYAL_DECREE` | 스토리 특수 (왕실 칙령) |

---

## 5. 제약 시스템 (Constraints: Cooldown / Stamina / Political Consequences)

### 5.1 자원 및 사용 제약

- 높은 stamina 또는 별도 자원 소모
- 연속 사용 불가
- 마법은 쿨타임 존재
- 한 RUN당 사용 횟수 제한 가능
- 특정 세력 평판 필요

### 5.2 남용 방지

- 과도한 사용 시 페널티 이벤트 발생
- 남용 시 정치적 파장 발생

### 5.3 정치적 반작용

마법 사용 시 다음과 같은 정치적 파장이 발생할 수 있다:

- 세력 의심도 증가
- 특정 세력과 적대 가능
- 지하세력 관심 상승

---

## 6. 전투에서의 마법 위치 (Magic in Combat)

마법은 전투 시스템 내에서 다음과 같이 정의된다:

- 기본 공격이 아님
- 특수 Action으로 정의
- 높은 stamina 또는 별도 자원 소모
- 보너스 슬롯 조건 강화/생성 가능
- 판세 전환용
- 적 AI 반응 변화 유발

---

## 7. 진행 시스템 (Progression System)

### 7.1 영웅 서사 강화

**초반:**
- 마법 없음
- 순수 전술/기량 기반

**중반:**
- 정치적 권한 획득
- 제한적 마법 접근 가능
- 특정 세력과 연계 시 사용 가능
- 대가가 따름

**후반:**
- 제한적 초자연 개입 가능

### 7.2 시스템 효과

- 성장 체감 명확
- 스토리와 자연스럽게 연결
- 밸런스 붕괴 방지
- 정치 드라마와 충돌 없음
