# Rewards and Progression Spec v1 (Canonical)

## 보상 및 성장 시스템 정본 스펙

------------------------------------------------------------------------

# 0. 확정 요약

-   보상 범위: 골드 + 아이템 + EXP(런 단위) + 평판/호감도
-   영구 보상: 골드/아이템/EXP/평판 (전부 영구)
-   전투 보상 지급: 전투 종료 즉시
-   전투 보상 방식: 랜덤 드랍 테이블 기반
-   드랍 방식: 보스 개별 롤 + 잡몹 전투 단위 롤 혼합
-   EXP: 런 단위 누적 (레벨은 런 단위 성장 구조)
-   레벨업 보상: 플레이어 행동 기반 스탯 성장
-   골드 사용처: SHOP + 강화/제작 + 외형/스탯 변경
-   인벤토리: 슬롯 제한
-   EVENT 보상: 고정 + 드랍 혼합
-   퀘스트 보상: 허브/EXIT에서 일괄 정산
-   패배: RUN 즉시 종료 + 대부분 보상 상실
-   도주: 골드 일부 손실 (-10% 권장)
-   보상 상한 존재 (폭주 방지)
-   보상 RNG는 전투 RNG와 분리

------------------------------------------------------------------------

# 1. 보상 카테고리

## 1.1 골드

-   주요 경제 자원
-   SHOP 구매
-   강화/제작
-   외형/스탯 변경(후속 확장)

## 1.2 아이템

-   장비
-   소비 아이템
-   퀘스트 아이템

## 1.3 EXP (런 단위 성장)

-   런 동안 누적
-   런 종료 시 리셋 가능 (메타 설계는 추후 확장)

## 1.4 평판/호감도

-   세력 평판
-   특정 NPC 호감도
-   이벤트/퀘스트 분기 조건에 영향

------------------------------------------------------------------------

# 2. 전투 보상 구조

## 2.1 지급 시점

-   VICTORY 즉시 지급

## 2.2 보상 방식

-   랜덤 드랍 테이블 기반
-   기본 골드 드랍 존재

## 2.3 드랍 롤 구조

-   잡몹: 전투 단위 1회 롤
-   보스/엘리트: 개별 롤

## 2.4 상한 규칙

-   전투당 최대 드랍 수 상한 존재
-   상한은 난이도/노드 타입에 따라 조정 가능

------------------------------------------------------------------------

# 3. EXP 및 성장 시스템

## 3.1 EXP 설계

-   런 단위 누적
-   레벨업 임계값은 외부 밸런싱 테이블에서 정의

## 3.2 레벨업 보상 구조

-   플레이어 행동 기반 성장 시스템

예시: - 근접 공격 위주 → ATK 성장 - 회피 중심 → EVA 증가 - 방어 위주 →
DEF 증가

> 성장 로직은 "행동 통계 기반 보정 테이블"을 외부에서 정의

------------------------------------------------------------------------

# 4. 인벤토리 시스템

## 4.1 슬롯 제한

-   기본 슬롯 수 고정 (예: 20)
-   초과 시 획득 불가 또는 선택 드랍

## 4.2 아이템 획득 실패 처리

-   SYSTEM 이벤트로 안내
-   플레이어 선택 유도 가능

------------------------------------------------------------------------

# 5. EVENT 및 퀘스트 보상

## 5.1 EVENT 보상

-   선택지 고정 보상 (서버 SoT)
-   일부 이벤트는 드랍 테이블 사용 가능

## 5.2 퀘스트 보상

-   퀘스트 완료 시 즉시 지급하지 않음
-   허브 또는 EXIT 노드에서 일괄 정산

------------------------------------------------------------------------

# 6. 패배 및 도주 처리

## 6.1 DEFEAT

-   RUN 즉시 종료
-   조력자 비용/서비스 비용/획득 보상 대부분 상실
-   영구 메타 자산 유지 여부는 별도 메타 스펙에서 정의

## 6.2 FLEE_SUCCESS

-   전투 보상 없음
-   골드 -10% 손실 권장
-   아이템은 유지

------------------------------------------------------------------------

# 7. 보상 RNG 정책

-   전투 RNG(seed+cursor)와 분리
-   보상 전용 RNG seed 사용
-   보상 RNG는 전투 결과와 독립적으로 관리

------------------------------------------------------------------------

# 8. 밸런스 가드레일

-   보상 상한 존재 (폭주 방지)
-   최소 보상 보장 없음
-   난이도 상승 시 드랍 확률 보정 가능

------------------------------------------------------------------------

# 9. 테스트 기준

-   동일 보상 RNG seed → 동일 드랍 결과
-   전투 승리 직후 보상 즉시 반영
-   슬롯 초과 시 획득 불가 처리 확인
-   도주 시 골드 10% 감소 확인
-   패배 시 RUN 종료 및 보상 상실 확인

------------------------------------------------------------------------

# 결론

본 문서는 v1 보상 및 성장 시스템의 정본 스펙이다. 세부 드랍 테이블, EXP
테이블, 아이템 정의는 외부 플랫폼에서 확정한다.
