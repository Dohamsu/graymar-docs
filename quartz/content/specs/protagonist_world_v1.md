# Protagonist & World v1

> 통합 문서: 왕국 스케일 + 단일 주인공 + 용병 출신 + 서사 톤 + 영웅 성장 구조
> 장르: 영웅 서사 중심 정치 갈등 RPG
> 설계 결론: "한 명의 무명 용병이 왕국의 정치 균형을 뒤흔드는 존재로 성장하는 이야기"

---

## 1. 왕국 구조

> 세력 상세: `political_narrative_system_v1.md` §1-2

왕국 = 수도(정치 중심) + 귀족 영지 3~5개 + 국경 요새 + 상업/농업 지역

허브 위치:
- 초기: 수도 인근 용병 집합소
- 챕터 진행: 수도 내부 / 귀족 영지 / 국경 군영

---

## 2. RUN 유형 (수도 / 지방 / 국경)

수도 RUN:
- 암살 방지
- 정보 수집
- 귀족 협상

지방 RUN:
- 반란 진압
- 세력 분쟁 개입
- 경제 분쟁 해결

국경 RUN:
- 기사단 지원
- 외부 위협 조사
- 밀수 차단

RUN은:
- 에피소드 단위
- 임무 단위
- 성장 단위

하지만 주인공은 동일 인물이다.

## 3. 긴장 확산

> 긴장도 5단계 모델: `political_narrative_system_v1.md` §5-6

---

## 4. 주인공 정체성: 단일 주인공 철학

### 4-1. 기본 철학 (세계는 리셋되지 않음)

- 플레이어는 한 명의 인물이다.
- 세계는 이 인물을 중심으로 변화한다.
- 세력 관계, 정치 균형, 스토리 기억은 모두 누적된다.
- 리셋은 RUN 단위이며, 세계는 리셋되지 않는다.

### 4-2. 서사적 효과 (NPC 기억, 배신의 무게)

단일 주인공이므로:
- NPC 기억 누적 가능
- 배신의 무게 증가
- 신뢰 축적의 의미 증가
- 영웅 신화 형성 가능

---

## 5. 용병 출신 배경

### 5-1. 초기 상태 (이름 없는 용병, 평판 0)

플레이어는:
- 이름 없는 용병
- 특별한 혈통 없음
- 정치적 영향력 없음
- 세력 소속 없음
- 신뢰도 0

### 5-2. 시작 위치 (용병 집합소)

허브:
- 용병 집합소
- 길드 등록
- 배급 장비 수령
- 낮은 등급 임무만 수락 가능

### 5-3. 전투적 정체성 (초기 → 성장)

초기에는:
- 기본 전투 능력
- 전술 숙련 낮음
- 보너스 슬롯 조건 엄격

성장하면:
- 전술 숙련 상승
- 보너스 슬롯 활용 증가
- 정치적 설득력 상승

### 5-4. 용병 출신의 장점

- 영웅 서사와 궁합 좋음
- 플레이어 투영 용이
- 선택의 의미가 커짐
- 세력 간 중립 시작 가능

## 7. 서사 톤 (무겁지만 희망 존재)

### 7-1. 기본 톤

- 정치 갈등은 존재하지만 절망 일변도는 아님
- 플레이어는 단순한 생존자가 아니라 "변수"
- 세계는 플레이어의 선택에 의해 변한다
- 비극 가능성은 있으나, 무력감 중심은 아님

### 7-2. 감정 구조 (초기 / 중반 / 후반)

플레이어가 느껴야 할 감정:

- 초기: 의심, 탐색
- 중반: 책임감, 긴장감
- 후반: 결단, 희생, 명예

### 7-3. NPC 설계 방향 (선악 이분법 지양)

NPC는 단순 악역이 아님:

- 각 세력은 나름의 명분 보유
- 대화로 설득 가능
- 배신은 개인적 동기 기반
- 선/악 이분법 지양

### 7-4. 전투 톤

전투는:
- 단순 살육이 아님
- 정치적 의미를 가진 충돌
- 암살/구출/저지 임무 다수

### 7-5. 허브 톤

허브는:
- 정치의 중심
- 정보의 교차점
- 음모와 협상이 오가는 공간
- 영웅의 선택이 기록되는 장소

### 7-6. LLM 서사 가이드

LLM은:
- 과장된 먼치킨 묘사 지양
- 감정과 정치적 맥락 강조
- 승리에도 비용이 있다는 느낌 유지
- 패배에도 명예가 존재하도록 표현

## 8. 영웅 성장 구조 (ACT 1-6)

무명 → 인정받는 용병 → 신뢰받는 중재자 → 정치 변수 → 권력 핵심 → 새로운 질서의 창조자

**ACT 1: 무명 용병**
- 용병/조사관
- 세력의 도구
- RUN 성공을 통해 인정 획득

**ACT 2: 인정받는 용병**
- 특정 세력의 신뢰 인물
- 정보 접근 가능
- 정치 사건 개입 시작

**ACT 3: 신뢰받는 중재자**
- 정치적 영향력 확보
- 세력 간 중재자
- 주요 인물 구출/설득

**ACT 4: 정치 변수**
- 암투의 핵심 변수
- 세력 균형 변화의 촉매

**ACT 5: 권력 핵심**
- 세력 균형을 좌우하는 인물

**ACT 6: 새로운 질서**
- 새로운 질서를 만드는 존재

이 변화는 다음에 의해 발생:
- RUN 성공
- 정치 사건 개입
- 세력 균형 변화
- 주요 인물 구출/설득

---

## 9. 엔딩 방향

엔딩은 단순 승패가 아니라:

- 어떤 질서를 선택했는가
- 누구를 희생했는가
- 어떤 세력이 남았는가
- 당신은 어떤 영웅으로 기록되는가

단일 주인공이므로:
- 세계 엔딩은 플레이어 중심
- 정치 질서 재편은 플레이어 선택 반영
- 여러 챕터가 이어져도 동일 인물
