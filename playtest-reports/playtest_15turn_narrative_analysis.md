## 2. 내러티브 일관성 평가

### 2.1 서사 흐름 (7/10)

**긍정적**:
- T1→T2: 선술집에서 의뢰 수락 → 시장 진입의 자연스러운 전환
- T4→T7: "관찰 → 대화 → 소문 → 종이 조각 → 비밀 통로"로 이어지는 점진적 발견 구조가 우수
- T12→T13: 미행 후 위협이라는 에스컬레이션이 자연스럽게 연결

**문제점**:
- **정보 반복**: "동부 창고", "비밀 통로", "장부"가 T5~T13까지 매 턴 반복. LLM이 새로운 정보를 추가하지 않고 같은 소재를 재탕
- **발견의 무효화**: T7에서 비밀 통로 위치를 이미 발견했는데, T11~T13에서 다시 "비밀 통로가 어디인지" 묻는 상황 발생
- **결말 급작스러움**: T13에서 갑자기 NATURAL 엔딩. 플레이어 관점에서 아직 탐색 중이었음

### 2.2 NPC 일관성 (4/10) — 심각한 문제

**NPC 이름 비공개 시스템 미작동**:
- 설계: FRIENDLY 1회 / CAUTIOUS 2회 / HOSTILE 3회 소개 후 이름 공개
- 현실: 12턴 동안 모든 NPC가 서술자("약초 노점의 노부인", "입이 가벼운 술꾼" 등)로만 호칭
- 엔딩에서 갑자기 11명의 NPC 실명이 등장 (로자, 미렐라, 에드릭 베일 등)
- **불일치**: 내러티브에서 한 번도 이름이 언급되지 않은 NPC들이 에필로그에서 등장

**NPC 등장 패턴 문제**:
| NPC 서술자 | 등장 턴 | 추정 실명 | 이름 노출 |
|-----------|---------|----------|----------|
| 약초 노점의 노부인 | T2,4,5,6,7,8,9 | 로자? | ❌ 없음 |
| 입이 가벼운 술꾼 | T4,5,6,8 | 불명 | ❌ 없음 |
| 수상한 창고 관리인 | T7,9,10,11,12,13 | 마이렐 단 경 | ❌ 없음 |
| 날카로운 눈매의 회계사 | T2,8 | 에드릭 베일? | T9에서 갑자기 이름 언급 |

- T9에서 "에드릭 베일의 모습은 보이지 않았으나"라는 문장이 등장하나, 이전에 이 이름이 소개된 적 없음
- "약초 노점의 노부인"은 7턴에 걸쳐 등장하면서도 이름이 없음 → 몰입 저하

**NPC 성격/태도 일관성**:
- "수상한 창고 관리인": T7에서 경계적 → T10에서 협조적으로 열쇠를 보여줌 → T13에서 위협에 쉽게 굴복. 태도 전환이 너무 급격
- "약초 노점의 노부인": 모든 턴에서 "조심하시오" 패턴 반복. 캐릭터 깊이 없음

### 2.3 맥락 유지 (5/10)

**긍정적**:
- 장소 분위기(밤, 바람, 파도 소리)가 전 턴에 걸쳐 일관
- 플레이어의 이전 행동을 다음 턴에서 이어받는 구조 (예: T6의 종이 조각 → T7에서 조사)

**문제점**:
- **이동 불가**: T8에서 "다른 장소로 이동한다" → IntentParser가 MOVE_LOCATION으로 파싱하지 못함. 내러티브가 "회계사가 막음"으로 처리했지만, 게임 메커니즘 차원의 문제
- **장소 고정**: 10턴 LOCATION 중 100% LOC_MARKET에서만 진행. 다양성 없음
- **종이 조각 추적 실패**: T6에서 노부인이 종이 조각을 줬는데, T7에서 이를 참조하나 T8 이후 완전히 잊힘

## 5. 개선 권장사항

1. **[Critical] structuredMemory 점검**: MemoryCollector/MemoryIntegration 서비스가 턴 처리 후 호출되는지 확인
2. **[Critical] NPC 이름 공개 파이프라인 점검**: NPC와의 상호작용 횟수 추적 → 이름 노출 조건 충족 시 LLM 프롬프트에 실명 주입
3. **[High] 정보 반복 방지**: LLM 프롬프트에 "이전에 공유된 정보 요약"을 포함하여 같은 소재 재탕 방지
4. **[High] MOVE_LOCATION 인텐트 파싱**: "다른 장소로 이동한다" 같은 입력을 MOVE_LOCATION으로 매핑
5. **[Medium] 이벤트 쿨다운**: 동일 이벤트 연속 매칭 방지 (최소 2턴 쿨다운)
6. **[Medium] 엔딩 조건 조정**: 최소 턴 수 보장 (예: 15턴 미만이면 NATURAL 엔딩 억제)
7. **[Low] NPC 성격 일관성**: 관리인이 경계→협조→굴복으로 전환되는 과정에 설득 판정 연동
